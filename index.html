Client Portal

A Web Portal for users to retrieve statements, to view royalties data and insights. This is a collaborative task where I was assigned to re-brand, support maintenance and enhancement of new features.

Technology

Portal is build on AngularJS framework and uses Node.js as a middleware for handling API request. The login template is written in Thymeleaf while the graphical view uses D3.js and DataMaps.

Analytics

An Android application based on Client Portal for users to access their royalties on the go. I was assigned to picked up an unfinished build for release and enhancement of new pipeline feature.

Technology

Analytics is build with Android Java and Retrofit HTTP Client.
